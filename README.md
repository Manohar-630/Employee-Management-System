# Employee Management System

## Overview

Employee Management System is a Python-based application that performs CRUD (Create, Read, Update, Delete) operations using MySQL. It helps manage employee records efficiently through a command-line interface.

---

## Features

- Secure Login Authentication
- Add Employee
- View Employees
- Search Employee
- Update Employee
- Delete Employee
- MySQL Database Integration

---

## Technologies Used

- Python 3
- MySQL
- mysql-connector-python
- Git
- GitHub
- Visual Studio Code

## Installation

1. Clone the repository

```bash
git clone https://github.com/Manohar-630/Employee-Management-System.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the SQL script

```
sql/employee_db.sql
```

4. Start the application

```bash
python src/main.py
```
